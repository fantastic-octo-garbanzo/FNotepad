1. Welche Dateiformate kann ich öffnen?
Mit FNotepad, können Sie  erstmal alle Dateiformate öffnen, aber es werden nicht alle Dateiformate lesbar dargestellt.

2. Ist FNotepad kostenlos?
Ja FNotepad ist kostenlos.

3. Wie öffne ich Dateien?
Klicken Sie auf Datei und dann auf Öffnen, oder drücken Sie die Tastenkombination Strg + O.

3. Wie Speichere ich Dateien?
Klicken Sie auf Datei und dann auf Speichern oder Speichern als..., oder drücken Sie die Tastenkombination Strg + S.

4. Wie erstelle ich eine neue Datei?
Klicken Sie auf Datei und dann auf neue Datei, oder drücken Sie die Tastenkombination Strg + N.

5. Wie exportiere ich eine Datei als PDF?
Klicken Sie auf Datei und dann auf Datei als PDF exportieren, oder drücken Sie die Tastenkombination Strg +  Y.

6. Wie exportiere ich eine Datei als HTML?
Klicken Sie auf Datei und dann auf Datei als HTML exportieren, oder drücken Sie die Tastenkombination Strg +  Y.

7. Wie erstelle ich ein neues Fenster?
Klicken Sie auf Datei und dann auf Neues Fenster, oder drücken Sie die Tastenkombination Strg + G.

8. Wie drucke ich eine Datei aus?
Klicken Sie auf Datei und dann auf Drucken, oder drücken Sie die  Tastenkombination Strg + P.
